# Excel to Mobile Image Skill

一个用于将工作计划 Excel 表格转换为适合手机阅读的、设计精美的莫兰迪风格“手机卡片流”长图的自动化工具。同时，它也会输出一份经过排版与格式美化的整理版 Excel。

---

## 🚀 主要功能

1. **自动清理与加工**：自动加载输入的 Excel，根据销售拼音自动排序，增加序号并整理字段列。
2. **Excel 自动美化**：使用深色 Slate-800 风格作为表头，并设置奇偶行斑马纹背景，自动适配最优列宽。
3. **高质感手机长图**：使用 Chrome Headless 渲染高对比度、无损画质（3倍视网膜缩放）的手机卡片流长图。
4. **智能自适应高度裁剪**：大图底部包含物理定位线，脚本在截图后自动从下往上扫描定位，精准进行像素级裁剪，彻底解决底部大片留白或内容截断问题。
5. **防进程冲突与卡死设计**：
   - 每次渲染采用独立的临时 `User Data Directory`，确保多实例并发及前后次运行时互不占锁。
   - 对 Chrome 渲染操作内置 **30 秒超时强退机制**，并捕获完整的运行日志以应对挂起异常。

## 🛠️ 环境依赖

### 1. Python 依赖库
在终端运行以下命令安装所需的 Python 库：
```bash
pip install pandas openpyxl Pillow pypinyin
```

### 2. 浏览器依赖
脚本截图依赖系统中的 Chrome 浏览器：
- 支持 **macOS** / **Linux** / **Windows**。
- 脚本会智能寻找您系统默认安装路径下的 **Google Chrome**、**Microsoft Edge** 或 **Chromium**。
- 如需自定义路径，可在环境变量中指定 `CHROME_PATH`。

---

## 📖 使用指南

在项目根目录下执行以下命令：

```bash
python3 scripts/excel_to_image.py --input 工作计划.xlsx
```

### 参数说明
- `-i, --input` (*必填*)：输入的原始工作计划 Excel 文件路径。
- `-o, --output-dir` (*选填*)：生成的产物输出目录，默认与输入文件处于同级目录。

### 生成产物
运行成功后，将在输出目录生成以下两个文件：
1. **`[输入文件名]_整理.xlsx`**：经过精美格式化排版后的表格文件。
2. **`[输入文件名]_手机卡片流.png`**：专为手机移动端适配的高清长图。
